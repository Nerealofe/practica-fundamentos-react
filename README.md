# Nerepop

Este proyecto es una aplicación hecha con React y TypeScript.

## Funcionalidades

- Iniciar sesión.
- Mantener la sesión iniciada.
- Ver una lista de productos.
- Ver el detalle de un producto.
- Crear nuevos productos.
- Borrar productos.
- Filtrar productos por nombre y precio.
- Añadir tags a los productos.
- Página 404 para rutas que no existen.

## Tecnologías utilizadas

- React
- TypeScript
- React Router
- Vite
- Fetch API

## Ejecutar el proyecto

Instalar dependencias:

```bash
npm install
```

Iniciar el proyecto:

```bash
npm run dev
```
